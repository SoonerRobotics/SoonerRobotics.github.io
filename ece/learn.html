Learning ECE

Core Classes for EE and CE

Both Computer Engineering (CE) and Electrical Engineering (EE) have major specific classes that are emphasized. However, there are shared classes between them. These core clases are listed below. In this wiki page are tips listed for each class as well as essential ECE knowledge to succeed in robotics.

  • Digital Design

    • Digital Design focuses on the basics of digital circuits. This includes number systems (like binary and hexadecimal), combinational logic circuits, sequential/combinational logic design, decoders/encoders, and more. This will be the first class you take with a lab portion. The lab will focus on the basics of building/debugging circuits.

  • Digital Signals and Filters

    • Digital signals and filtering focuses on the basics of digital signals and filtering. You will learn about the core techniques for modeling signals and systems using math, how to perform the analysis of these signals and systems, and how to design simple filters to process these signals. This class will also provide a basic introduction to MATLAB.

  • Circuits 1

    • Circuits 1 focuses on the basics of circuit analysis. This includes fundamentals such as Kirchoff's Voltage Law, Kirchoff's Current Law, Node Voltage Analysis, Mesh Current Analysis, Voltage/Current Dividers, Source Transformations, Equivalent Circuit Models, Capacitors/Inductors (DC Response), Phasors, and an introduction to AC Circuit Analysis and Complex Power. The material learned in this class will be your toolbox for the rest of the ECE classes, so make sure to learn this.

  • Circuits 2

    • Circuits 2 focuses on continuing knowledge about circuit analysis principles. This includes analysis of circuits in time and frequency domains, two port newtorks, impulse response, convolution, differential equations, different transforms. This class emphasizes analysis with the use of the MATLAB.

  • Introduction to Electronics

    • Intro to electronics focuses on circuit analysis with respect to specific components. You will learn about operational amplifiers, diodes, BJTs, MOSFETs, and using these components to build amplifiers. This course builds off of circuits 1 analysis with new components.

  • Circuits Lab

    • Circuits Lab focuses on implementing circuit analysis knowledge from circuits 1 to design and construct circuits. This course teaches basic instrumentation use and debugging techniques. This class also teaches PCB work with the use of Eagle.

  • Microprocessor System Design

    • Microprocessor system design focuses on microprocessor/microcontroller architecture. This is achieved by covering topics on number systems, bus interfacing, programming languages (C and simple assembly), I/O operations (serial and parallel devices), using interrupts in hardware/software, generating waverforms, analog to digital/digital to analog conversions, and register access.

  • Signals and Systems

    • Signals and systems focuses on learning techniques (math and tools) to analyze linear continuous/discrete time systems. By the end of the class you will be able to set-up input/output relationships for linear systems (in both time and freq domains).

  • Electronics Lab

    • Electronics lab focuses on circuit design, debugging, and advanced instrumentation skills. ELab also focuses on circuit reliability, independent design projects, and technical writing. This lab allows you to continue on from intro to electronics and circuits 2 and implement skills learned.

Buck Convertors

Buck convertors are step down convertors designed to oppose or restrict an input current which can be used to create voltages or currents lower than the input. They are used when the DC output voltage needs to be lower than the DC input voltage. They implement switched-mode power supply (SMPS) which turns power on and off rapidly. This results in an average output between the off and on. Buck convertors use a PMOS MOSFET as the switch. Buck convertors also use inductors to limit voltage peaks as part of the energy is stored in the magnetic field, when the switch is open the magnetic field collapses which allows current to flow.

L298N is a standard buck convertor that is used by SCR. You can read more about this buck convertor here.
